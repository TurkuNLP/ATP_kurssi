### 1. lecture
* General course information

### 2. lecture

* Reading manuals

* materials: [Unix_1.ipynb](Unix_1.ipynb), [Unix_2.ipynb](Unix_2.ipynb)

### 3. lecture

* materials: [Unix_4.ipynb](Unix_4.ipynb)
